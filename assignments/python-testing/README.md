# 📘 Assignment: Python Testing

## 🎯 Objective

Learn how to write automated tests for Python code using the standard library `unittest` module. Students will practice checking expected behavior, catching edge cases, and using tests to confirm that code works as intended.

## 📝 Tasks

### 🛠️ Write Basic Unit Tests

#### Description
Create a test file that checks a few simple Python functions with `unittest`.

#### Requirements
Completed program should:

- Import the functions from the provided starter code
- Write tests that verify correct results for at least three functions
- Include at least one test that checks an edge case or unusual input
- Run successfully with Python's built-in `unittest` runner


### 🛠️ Find and Fix a Bug with Tests

#### Description
Use your tests to identify a broken function in the starter code, then update the code so the tests pass.

#### Requirements
Completed program should:

- Identify at least one failing test before making changes
- Fix the buggy function in the starter code
- Re-run the tests and confirm they all pass
- Keep the function behavior clear and easy to understand


### 🛠️ Add One More Test Case

#### Description
Expand the test suite with one extra case that protects against a mistake a future programmer might make.

#### Requirements
Completed program should:

- Add one additional test beyond the minimum required tests
- Cover either a boundary value or a common error case
- Use a descriptive test method name
- Help explain why the code should behave that way
